# Project Structure (Step 1)

## Chosen stack (simple and beginner-friendly)

- **HTML** for structure
- **CSS** for styling
- **Vanilla JavaScript** for behavior
- **JSON file** for starter product data

No build tools, no framework, and no database yet.

## Files created

- `index.html` — app shell with placeholder sections for dashboard + products
- `styles/main.css` — base styles and simple card layout
- `scripts/app.js` — tiny starter script proving JS is connected
- `data/products.json` — one example product record with all required fields
